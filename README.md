# 🎮 Hangman Game

A simple **terminal-based Hangman game** built using Python.

## 📌 Description

Hangman is a classic word guessing game where the player tries to guess the hidden word letter by letter before running out of attempts.

This project was built to practice Python programming concepts such as functions, loops, conditions, file handling, and input validation.

## ✨ Features

* Random word selection from an external file (`words.txt`)
* Letter guessing system
* Tracks guessed letters
* Limited number of attempts
* ASCII Hangman drawing
* Hint system
* Score calculation
* Input validation to handle incorrect user inputs

## 🛠️ Technologies

* Python 3
* File Handling
* Basic Game Logic

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/joodshatnawi/Hant_game.git
```

2. Navigate to the project folder:

```bash
cd Hant_game
```

3. Run the game:

```bash
python hangman_game.py
```

## 🎯 How to Play

1. Choose a word from the available list.
2. Guess letters one by one.
3. Correct guesses reveal the hidden word.
4. Wrong guesses reduce your remaining attempts.
5. Use hints carefully because they cost an attempt.
6. Try to guess the word before the hangman is completed!
